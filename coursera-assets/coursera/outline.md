# Data Engineering with Databricks

**Duration**: 3 weeks

## Overview

A comprehensive course covering DLT Fundamentals and Medallion Architecture.

## Learning Outcomes

1. Apply foundational concepts to real-world scenarios
2. Analyze complex systems and identify optimization opportunities
3. Evaluate trade-offs in design decisions
4. Create end-to-end solutions using industry best practices

---

## Week 1: DLT Fundamentals

Covers Course overview and learning outcomes, Delta Live Tables as the core technology, Hands-on labs with Databricks Community Edition, What Delta Live Tables solves, and Declarative vs imperative ETL.

**Learning Objectives**:

- Remember core terminology and foundational concepts
- Analyze the relationships between key system components
- Apply basic techniques to introductory problems

## Week 3: Capstone Project

Build a complete medallion architecture pipeline using Delta Live Tables on the Databricks FREE edition. This project brings together all concepts from the course: DLT pipelines, expectations, streaming, bronze/silver/gold layers, and Change Data Capture.

**Learning Objectives**:

- Create an end-to-end solution integrating all course concepts
- Evaluate the solution against production quality standards
- Synthesize learnings from previous modules into a cohesive project
